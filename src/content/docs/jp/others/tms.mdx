## 歴史

<Tabs>
<TabItem label="ハード, NG - Wii U">
<Steps>
1. 幻影異聞録♯FEは2017年10月16日に発売されました。
2. 2017年05月3日に最初の走りがO-denに完走されました。[**11:12:55**](https://www.nicovideo.jp/watch/sm31141771)。
3. 2018年04月29日に、legrandgrandさんがHeroMysticさんのチャートを充実して世界記録を更新しました。 [**11時間15分36**](https://www.youtube.com/watch?v=OVC-2SQk8iM)
4. 2018年05月6日に,　HeroMysticさんは記録を取り返した。[**10時間30分45**](https://www.twitch.tv/videos/258238938)。
5. 2018年05月9日に,　legrandgrandさんはまた記録を更新しました。[**9時間12分15](https://youtu.be/noBNKezb4TI)。
6. 現在の世界記録はlegrandgrandさんの[**8時間52分42**](https://youtu.be/HIhQ-qu3Oi8)。
</Steps>
</TabItem>

2. 最初の完走したタイムはlegrandgrandさんの[**8時間29分58**](https://www.youtube.com/watch?v=qwoLPWnImPg).
