## Speedrun History
<Steps>
1. Raidou Kuzunoha vs. the Soulless Army was released in Japan on August 30, 2012 and worldwide by September 26, 2013.
2. The first ever recorded run was on 2016/05/30 by 運の香, read as UnnoKou, with a time of [**5:11:12**](https://www.nicovideo.jp/watch/sm28952413) using the PS2 version.
3. ExcelitoL improved the time using a PS3 console and the North American PS Store version throughout 2021.
4. The current world record was done by ExcelitoL with a time of [**4:31:22**](https://www.youtube.com/watch?v=OKySAA-HPxc).
</Steps>

## Glitches & Tricks
<details>
<summary>Vsync Delay OFF (PSN only)</summary>
By using the PS3's PS2 emulation debug menu you can turn off Vsync delay. This makes the infamous PSN port run at a much smoother frame rate and is currently allowed for speedruns.
</details>
