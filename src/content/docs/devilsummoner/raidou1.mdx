## Speedrun History
<Steps>
1. Raidou Kuzunoha vs. the Soulless Army was released in Japan on August 30, 2012 and worldwide by September 26, 2013.
2. The first ever recorded run was on 2016/05/30 by 運の香, read as UnnoKou, with a time of [**5:11:12**](https://www.nicovideo.jp/watch/sm28952413).
3. Runner ToxicTT did the first ever run in English on 2022/10/28 with a time of [**5:30:44**](https://www.twitch.tv/videos/1636776368)
4. The current world record was done by ExcelitoL with a time of [**4:31:22**](https://www.youtube.com/watch?v=OKySAA-HPxc).
</Steps>
