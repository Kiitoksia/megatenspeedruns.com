## Speedrun History
<Steps>
1. Digital Devil Story: Megami Tensei was released in Japan on September 11, 1987.
2. The first recording of a speedrun found was uploaded by [めし太郎](https://www.nicovideo.jp/user/2729422?rf=nvpc&rp=watch&ra=video_information) on 2011/07/06 with a time of [**9:16:03**](https://www.nicovideo.jp/watch/sm14942670)
3. On 2013/04/05, biim uploaded a run with a time of [**5:04:05**](https://www.nicovideo.jp/watch/sm20530524).
4. On 2013/07/29, runner sinri uploaded a new record with a time of [**4:25:02**](https://www.nicovideo.jp/watch/sm21472094).
5. On 2014/0215, runner dasi uploaded an improvement to the world record with a time of [**3:18:06**](https://www.nicovideo.jp/watch/sm22899800).
6. The current world record seems to be a time of [**3:13:04**](https://www.twitch.tv/videos/2357513079?collection=RjiGK9oJcBZ6PQ) on 2025/01/19 by Goro_Itsumi . This run uses *Medical Loop*.
</Steps>
