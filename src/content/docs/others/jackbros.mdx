## Speedrun History

<Tabs>
<TabItem label="Beat the Game - Jacks">
<Steps>
1. Jack Bros. was released in Japan on September 29, 2005.
2. On 2017/01/08, the first ever recorded run was done by SomeGirl with a time of [**25:35**](https://youtu.be/K15Dcpv4vRw)
3. On 2017/04/21, Witchs_Hex achieved a new world record with a time of [**23:41**](https://youtu.be/ZbuK4J1MMoU)
4. 3 years later, on 2020/11/21, swordsmankirby beat the world record with a time of [**23:23**](https://youtu.be/PhbqjHWz2AY)
5. The current world record is held by SmoothColaPepsi with a time of [**22:34**](https://youtu.be/i8wqf9478tk)
</Steps>
</TabItem>
