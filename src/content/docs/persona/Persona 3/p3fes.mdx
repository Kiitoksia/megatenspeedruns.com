## Speedrun History
<Tabs>
    <TabItem label="The Journey">
        <Steps>
            1. Shin Megami Tensei: Persona 3 FES was released in Japan on April 19, 2007.
            2. The first recorded run was done by newby24 with a time of [**43:49:01**](https://www.twitch.tv/videos/157810640).
            3. 1 year later, on 2018/07/31, Darkkefka beat the world record with a time of [**23:12:09**](https://youtu.be/uLJzJFE-MDU)
            4. On 2020/05/24, Keeanolookalike brought the time down further to [**21:55:03**](https://www.twitch.tv/videos/630455571)
            5. On 2021/10/23, Darkkefka set a new record with a time of [**20:17:44**](https://youtu.be/VPmV0vBxJXs)
            6. On 2022/09/04, TatsuyaAct achieved the first sub 20h with a time of [**19:50:47**](https://youtu.be/bxJ3uwRKp9k)
            7. On 2023/02/26, Fenr1r79 set a new record with a time of [**19:26:33**](https://www.twitch.tv/videos/1749735432)
            8. The current world record holder is Darkkefka with a time of [**19:23:14**](https://www.twitch.tv/videos/1963205251)
        </Steps>
    </TabItem>
    <TabItem label="The Answer">
        <Steps>
            1. Shin Megami Tensei: Persona 3 FES was released in Japan on April 19, 2007.
            2. The first recorded run was done by lilfatkid with a time of [**13:56:24**](https://www.twitch.tv/videos/257118140). He would improve the record down to a time of [**9:09:39**](https://youtu.be/qt9UkUCJ-aM)
            3. On 2019/07/22, Flutter set a new record with a time of [**9:08:15**](https://www.twitch.tv/videos/457225926)
            4. 24 days later, lilfatkid reclaimed the world record with a time of [**8:54:42**](https://youtu.be/gNoJi3T9DWE) which he improved until 2021/11/05 when he finally achieved the first sub 8h run with a time of [**7:54:26**](https://youtu.be/0RKa5ezqGlU)
            5. On 2022/09/21, Kap was the second person to ever achieve a sub 8h time and set a new record with a time of [**7:48:47**](https://youtu.be/cU0C2-jz2dA)
            2. The current world record holder is Kap with a time of [**7:21:37**](https://youtu.be/hy9BOzJkwIA)
        </Steps>
    </TabItem>
</Tabs>
