## Speedrun History
<Steps>
1. Shin Megami Tensei IV Apocalypse released worldwide by December 9. 2016.
2. The first available recorded run was done by ChampionBeef on 2017/09/03 with a time of [**10:54:46**](https://youtu.be/SaqQ3u_mOaE)
3. 7 months later, fellow speedrunner and router Koolzo achieved a new world record with a time of [**9:53:50**](https://www.twitch.tv/videos/254592733) that he would later improve to an almost sub 7 time of [**7:09:42**](https://www.twitch.tv/videos/1024295352) on 2021/05/16, showcasing the potential of this run.
2. The current world record is held by Freedom_Pulse with a time of [**7:05:06**](https://www.twitch.tv/videos/1499712873). The potential for a sub 7h run is still up for grabs.
</Steps>
